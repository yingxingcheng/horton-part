From Command Line

In this tutorial, all calculations are performed via command line.

Step 1: Generate Molecular Density and Grid

First, one needs to generate molecular density and grid:

part-gen --inputs ./data/h2o.fchk --outputs h2o_dens.npz

To execute command in notebook, one needs to load os package.

[1]:

import os

Then, one can execute command through calling os.system function. The result is restored in co_dens.npz that is a Numpy npz file, which is also the input file of next step.

[2]:

os.system("part-gen --inputs ./data/h2o.fchk --outputs h2o_dens.npz")

[2]:

0

Step 2: Do Partitioning

Then, we employ the part-dens command to partition molecular density into atomic contributions. In this process, we utilize the local LISA scheme, which addresses a fixed-point problem through a self-consistent method. For additional information, refer to LisaDIISWPart.

It should be noted that, for now, only ISA schemes are available via the command line. For users interested in other partitioning schemes, please refer to the Examples tutorials.

The command line syntax is as follows:

part-dens h2o_dens.npz -t lisa --solver 201 --log INFO --output h2o_part.npz

The h2o_dens.npz is obtained from part-gen command and the output of the result is restored in h2o_part.npz file.

Again, we use the os.system function to execute the command line within the notebook.

[3]:

os.system(
    "part-dens --inputs h2o_dens.npz -t lisa --solver sc --log_level INFO --outputs h2o_part.npz"
)
